# Курс по Го в ШАД

Для работы с кодом нужен go 1.18 или выше.

```sh
git clone https://gitlab.atp-fivt.org/courses-public/golang/golang-2024s/go.git
cd go
go test ./sum/...
```

Проект можно открыть в любой go IDE с поддержкой модулей.
